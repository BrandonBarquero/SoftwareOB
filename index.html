Software OB

Software OB es una empresa costarricense con servicios de consultoría tecnológica y desarrollo de proyectos de soluciones de software a la medida.

Servicios

Diseño Web

Ponemos a su disposición nuestro equipo de diseñadores y desarrolladores para realizar su sitio web totalmente a medida.

Desarrrollo de aplicaciones móviles

Si su proyecto requiere el desarrollo de una aplicación móvil nativa, contamos con programación de aplicaciones en las plataformas Android y Apple.

Aseguramiento de calidad

Procuramos un proceso ágil para nuestros clientes, desde la Revisión de Requerimientos hasta la Entrega del Proyecto. Trabajamos bajo la metodología SCRUM.

Modelaje de procesos

Se proporciona un modelo de sistema que se construirá o mejorará, bajo uno de los estándares acordados con el cliente.

Nuestras tecnologías

En Software OB hacemos programación web con tecnologías libres destacadas por su estabilidad, amplio uso y desarrollo continuo. Así, para la puesta en producción de los sistemas que ofrecemos nos especializamos en Apache Web Server, Apache Tomcat e Internet Information Services (IIS). En bases de datos, nuestros desarrolladores se especializan en los gestores Mysql, Oracle y SQL server. Utilizamos Git como controlador de versiones, para garantizar un desarrollo rápido y seguro.



Usamos Boostrap para el desarrollo web por ser uno de los frameworks más utilizados debido a su estilo y simpleza al programar, así como los estándares de HTML5 y CSS3 por nuestro compromiso de brindar a nuestros clientes aplicaciones innovadoras con tecnologías actuales y funcionales. Por otro lado para el desarrollo de sitios web utilizamos los lenguajes de programación Java y ASP .Net C# como lenguajes principales combinados con JavaScript y JQuery



Para el desarrollo móvil utilizamos Java para aplicaciones en Android y C# en IOS.
